# Robot-Trajectory---BFS-Pathfinding
This project simulates autonomous robot navigation by finding the shortest collision-free path between two points on a grid-based map. The environment includes randomly generated obstacles ("contours") representing walls, barriers, or no-go zones.
🔑 Key Features

    Grid-based environment: Configurable n x n map with traversable cells and obstacles
    Dynamic obstacle generation: Random square-shaped contours simulate real-world barriers
    BFS pathfinding: Guarantees the shortest path in unweighted grids
    Interactive CLI: Real-time input for grid size, start/end coordinates
    Visual output: ASCII-based grid rendering with path visualization
